# destruction-derby

Destruction Derby is an internal CS:GO cheat.

# Features
- triggerbot
- no flash

# How to use
Inject compiled DLL into a CS:GO process (do some research into it). Once injected, press F8 to toggle no flash or F9 to toggle triggerbot
